JavaScript Syntax

JavaScript syntax is the set of rules, how JavaScript programms are constructed: 

            
                // How to create variables
                var x;
                let y;

                // How to use variables
                x = 5
                y = 6
                let z = x + y

JavaScript Values

The JavaScript syntax defines two types of values:

  • Fixed values
  • Variable values

Fixed values are called lieterals
Variable values are called variables

JavaScript Literals

The two most important syntax rules for fixed values are:
1. Numbers are written with or eithout decimals:
10.50 or 1001
2. strings are text, written within double or single quotes:
"John Bach" or 'John Bach'

JavaScript Variables

In a programming language, variables are used to store data values.
JavaScript uses the keywords var, let and const to declare variables.
An equal sign is used to assign values to variables.
In this example, x is defined as a variable. Then, x is assigned (given) the value 6: 

            
                let x;
                x = 6;

JavaScript operators

JavaScript uses arithmetic operators (+ - * /) to compute values: 

            
                (5 + 6) * 4;

JavaScript uses an assignment operator = ro assign values to variables. 
            
                let x, y;
                x = 4;
                y = 0;

JavaScript Expressions

An expression is a combination of values, variables and operators which computes to a value
The computation is called an evaluation.
For example, 5 * 10 evaluates to 50: 

            
                5 * 12
Expressions can also contain variable values: 
            
                x * 10;

The values can be of various types, such as numbers and strings
For example, "John" + "" + "Doe", evaluates to "John Doe" 
            
                "john" + "Doe" = "John Doe"
