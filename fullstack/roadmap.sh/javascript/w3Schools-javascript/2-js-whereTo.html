The script tag

In html js code is inserted between script and /script tag

        
            script
                document.getElementById('demo').innerHTML = 'My first js';
            script

JavaScript functions and events

A js function is a block of js code that can be executed when called for

For example, a function can be called when an event occurs, like when the user clicks a button.

            
                function soma(a, b) {
                    return a + b;
                }

JavaScript in head or body?

You can place any number of scripts in an html document

Scripts can be placed in the body or in the head section of an html page or in both.

JavaScript in head

In this example a js function is placed in the head section of an html page. 

            
                <html>
                <head>
                    <script>
                        console.log('Iae mano, como vai você?')
                    </script>
                </head>
                <body>
                </body>
                
                </html>

JavaScript in body

In this example a js function is placed in the body section of an html page. 

            
                <html>
                <head>
                </head>
                <body>
                    <h1>Meu formulário</h1>
                    <script>
                        console.log('Iae mano, como vai você?')
                    </script>
                </body>
                
                </html>
Note: Placing scripts at the bottom of the body element improves the display speed, because script interpretation slows down the display.

External JavaScript

Scripts can also be placed in external files

  • Extenal scripts are practical when the same code is used in many different web pages
  • JavaScript files have the file extension .js
  • To use an external script, put the name of the script file in the src attribute of a script tag

Example:

<script src="./myscript.js"></script>
    External JavaScript advantages
  • It separates html and code
  • It makes html and JavaScript easier to read and maintain
  • Cached JavaScript files can speed up page loads

To add several script files to one page use several script tags

            
                <script src="file1.js"></script>
                <script src="file2.js"></script>
                <script src="file3.js"></script>

External references

An external script can be referenced in three different ways:

  1. With a full URL (a full web address)
  2. With a file path (like /js/)
  3. Without any path

This example uses a full URL to link to myscript.js

            
                <script src="https://www.w3schools.com/js/myScript.js"></script>

This example uses a file path to link to myscript.js

            
                <script src="./js/myScript.js"></script>

This example uses no path to link to myscript.js

            
                <script src="myScript.js"></script>
