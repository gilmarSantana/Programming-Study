JavaScript Introduction

JS Can change html content

One of many js html methods is "getElementById()"
The example below "finds" an html element (with id="demo") and changes the element content (innerHTML) to "Hello JavaScript" 

            
                double quotes
                document.getElementById("demo").innerHTML = "Hello JavaScript"
                single quotes
                document.getElementById('demo').innerHTML = 'Hello JavaScript'

JS Can change html attribute values

In this example js changes the value of the src (source) attribute of an img tag 

            
                    img src="./img1.jpg" alt="" id="img1"
                    button onclick="getElementById('img1').src='./img2.jpg'">Change img button

JS Can change html css styles (css)

Changing the style of an html element is a variant of changing an html attribute 

            
                    document.getElementById('demo').style.fontSize = '35px';
                    document.getElementById('demo').style.color = 'red';

JS Can hide html css styles

Hiding html elements can be done by changing the display style 

            
                    document.getElementById('demo').style.display = 'none';

JS Can show html css styles

Showing html elements can also be done by changing the display style 

            
                    document.getElementById('demo').style.display = 'blockd';
