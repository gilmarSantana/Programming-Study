CSS RGB Colors

An rgb color value represents RED, GREEN, and BLUE light sources.

RGB Value

Is css, a color can be specified as an rgb value using this formula:
rgb(red, blue, green)

Each parameter defines the intendity of the color between 0 and 255.
For example, rgb(255,0,0) is displayed as red, because red is set to its highest value 255 and the others are set to 0.
To display black, set all color parameters to 0, like rgb(0, 0, 0)
To display white, set all color parameters to 255, like rgb(255, 255, 255).

RGBA Value

RGBA color values are an extension of rgb color values with an alpha channel - which specifies the opacity for a color.
An rgba color value is specified with:
rgba(red, green, blue, alpha)
The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all).

RGB é um acrônimo para "Red, Green, Blue" (Vermelho, Verde, Azul). É um sistema de cores aditivo que é amplamente utilizado para representar cores em dispositivos eletrônicos, como monitores de computador, TVs e câmeras digitais. No modelo RGB, cada cor é representada por uma combinação de intensidades de vermelho (Red), verde (Green) e azul (Blue). Cada componente de cor varia de 0 a 255, onde 0 representa a ausência da cor e 255 representa a intensidade máxima da cor. Ao combinar diferentes intensidades dessas três cores primárias, podemos obter uma ampla gama de cores. Por exemplo: RGB(255, 0, 0) representa a cor vermelha pura, com intensidade máxima de vermelho e nenhuma intensidade de verde e azul. RGB(0, 255, 0) representa a cor verde pura, com intensidade máxima de verde e nenhuma intensidade de vermelho e azul. RGB(0, 0, 255) representa a cor azul pura, com intensidade máxima de azul e nenhuma intensidade de vermelho e verde. Ao combinar diferentes valores de intensidade de vermelho, verde e azul, podemos criar uma ampla variedade de cores visíveis no espectro RGB.

RGBA é uma extensão do modelo de cor RGB, com o acréscimo de um quarto componente, o "A" que representa o canal alfa (alpha). Enquanto o RGB define a intensidade de vermelho, verde e azul para representar cores, o RGBA inclui o canal alfa para representar a transparência. No modelo RGBA, cada componente de cor (vermelho, verde, azul) varia de 0 a 255, da mesma forma que no modelo RGB. o canal alfa normalmente varia de 0 a 1, não de 0 a 255. O valor 0 indica transparência total (totalmente transparente), enquanto o valor 1 indica opacidade total (totalmente opaco). Essa é a convenção mais comummente utilizada em muitas linguagens de programação e softwares gráficos. A ordem dos componentes no modelo RGBA é a seguinte: RGBA(red, green, blue, alpha). Os valores são especificados em números inteiros separados por vírgulas, e normalmente são representados em um intervalo de 0 a 1 ou de 0 a 100% em algumas linguagens e softwares. Por exemplo: RGBA(255, 0, 0, 0.5) representa a cor vermelha com uma transparência de 50%. RGBA(0, 255, 0, 0.75) representa a cor verde com uma transparência de 75%. RGBA(0, 0, 255, 1) representa a cor azul totalmente opaca. O canal alfa permite que objetos e cores sejam sobrepostos e combinados com diferentes níveis de transparência, criando efeitos de camadas e transparências em elementos visuais. O modelo RGBA é comumente utilizado em gráficos, design web e em aplicativos que requerem controle preciso da transparência dos elementos visuais.

CSS HEX Colors

A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.

HEX Value

Sometimes you will see a 3-digit hex code in the CSS source. The 3-digit hex code is a shorthand for some 6-digit hex codes. The 3-digit hex code has the following form:
#rgb
Where r, g, and b represent the red, green, and blue components with values between 0 and f.

CSS HSL Colors

HSL stands for hue (matiz), saturation (saturação) and lightness (leveza)

HSL Value

In css a color can be specified using hue, saturation and lightness in the form:
hsl(hue, saturation, lightness)
Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green and 240 is blue.
Saturation is a percentage value. 0% means a shade of gray, and 100% is the full color.
Lightness is also a percentage. 0% is black, 50% is neither ligth or dark, 100% is white.
